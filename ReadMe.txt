To run the code you need to open a new terminal and then type 'python app.py' and hit enter.
Then open your browser and paste this URL 'http://127.0.0.1:5000' and then hit enter. 

Libraries Required :
Flask: This includes the submodules Flask, render_template, request, and jsonify.
You can install Flask by running the following command:
pip install Flask.
